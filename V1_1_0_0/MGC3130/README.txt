
**********************************************************************************
MGC3130 Library:

MGC3130 is a python library used to interact with MGC3130 gestic tecnology.
They were defined a lot of functions to interact with MGC3130 hardware

**********************************************************************************


**********************************************************************************
Requirements:

To use this library is necessary to have a "ArdUnoR3_RaspPiB+_GE_100x50 Rev. 1.0" hardware
distributed by Futura Group srl or a compatible hardware.

**********************************************************************************


**********************************************************************************
Install:

To install the library use the command:

sudo python setup.py install

**********************************************************************************


**********************************************************************************
How to import library in your sketch? At the head of the file it is necessary
write this code

import sys
sys.path.insert(0, "/usr/local/lib/python2.7/dist-packages/MGC3130"")
import MGC3130

**********************************************************************************
